# Just My Scent — Unofficial Redesign Concept

A custom headless Shopify storefront concept for **Just My Scent**, focused on editorial luxury, personal scent discovery, and accessible fragrance oils.

## Disclaimer

This is an **unofficial redesign concept** created for educational and portfolio purposes. This project is **not affiliated with or commissioned by Just My Scent**.

## Technology Stack

- [Next.js](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [ESLint](https://eslint.org/)
- [next/font](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) (Google Fonts)

## Local Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Other Commands

```bash
npm run lint   # Run ESLint
npm run build  # Create a production build
npm run start  # Serve the production build
```

## Current Phase

**Phase 1 — Design Foundation**

- Next.js application scaffold in the repository root
- Brand typography, color system, and homepage layout
- Announcement bar, header, hero, and disclaimer
